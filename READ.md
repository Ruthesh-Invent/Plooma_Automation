
Playwright Automation Framework

Here is our Playwright Automation Framework to automate our test cases. This Framework is created using Java Playwright.


## Features

Supports Chromium, Firefox, and WebKit browsers.
Robust Framework which can handle vast test scripts.
Supports parallel test execution and multiple contexts.

## Prerequisites

JDK 11 or higher

Integrated Development Environment

Maven

Basic Understanding of Playwright

## Installation

Install my-project

**Clone the repository:**

   ```bash
   git clone https://finons.visualstudio.com/DefaultCollection/BotmindsWeb/_git/BMWebAutomation
   cd playwright-framework
```
    
## Reports

In this framework we have used used ExtentReports to attach reports you can utilize Listeners class and can add the logs to the reports.
